Comandos Básicos de Git: add, commit y log

https://platzi.com/home/clases/11059-gitgithub/71786-comandos-basicos-de-git-add-commit-log/ 
        
            # iniciar un repositorio
            git init

            # agregarlo al área de staging
            git add archivo.txt
            git add .

            # to unstage:
            # Quita el archivo del staging, pero conserva los cambios en el working directory.
            git restore --staged archivo.txt
            # Deja de rastrear el archivo en Git, pero lo mantiene en el sistema de archivos.
            git rm --cached archivo.txt
            # Elimina el archivo del sistema de archivos y de Git.
            git rm --force archivo.txt

            # registrar los cambios en el repositorio
            git commit -m "Mensaje del commit"
            # modificar el mensaje del último commit
            git commit --amend -m "Nuevo mensaje del commit"

            # mostrar el historial de commits
            git log
