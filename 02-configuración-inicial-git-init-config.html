Configuración Inicial de Git: init y config

Saber la versión de tu Git

        
 git --versión

Preparar el primer proyecto con Git

        
mkdir curso-git-github-platzi
cd curso-git-github-platzi/

git init

# Si prefieres la rama principal como “main”:
    git config --global init.defaultBranch main.
# Actualiza la rama en el proyecto actual
    git branch -m main

# Personalizar tu configuración:
    git config --global user.name "MarcosSarLo"
        git config --global user.email "msl.net@gmail.com"

# revisar tu configuración:
git config --list

# recordar sintaxis de comandos:
    git --help
    git --help commit

Para empezar a trabajar con Git, primero debemos configurar nuestro nombre de usuario y correo electrónico. Esto es importante porque cada vez que hagamos un commit, esta información quedará registrada en el historial de cambios.

Para configurar nuestro nombre de usuario y correo electrónico, utilizamos los siguientes comandos:

        
            git config --global user.name "Tu Nombre"
            git config --global user.email "
